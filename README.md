## Description

A Nestjs and Typescript Bitcoin stratum mining server.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production build
$ npm run build

# production server
# *pre-req install pm2 (https://pm2.keymetrics.io/)
$ pm2 start dist/main.js
```



## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
